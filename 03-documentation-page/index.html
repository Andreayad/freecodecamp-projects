Introduction

CSS is the language that we use to style an HTML document. It stands for Cascading Style Sheets.

Three Ways to Insert CSS

  1. Inline - used to apply a unique style for a single element. <p style="color:green;">This is a paragraph</p>
  2. Internal - used if one single HTML page has a unique style. It is defined inside the <style> element, inside the head section. <style>
       p { background-color: black; }
    </style>
  3. External - change the look of an entire website by changing just one file. It must include a reference to the external style sheet file inside the element, inside the head section. <link rel="stylesheet" href="mystyle.css">
Syntax
<p> - Selector
{
   display:inline-block;
   color: blue;
} - Declaration Block
display - Property Nameinline-block - Property Value

Selector directs you to the HTML element that you want to style.

Declaration Block compose of one or more declaration separated by semicolons. It includes a CSS property name and a value, separated by a colon.

Element selector

A selector based on elements name.


Example: h1 p ol li ul section nav
Id selector

It uses the id attribute of an element to select specific element. It is used to select one unique element. To select an element write a hash(#) character followed by an id name.


Example: <p id="prgrph1">This is a first paragraph.</p> #prgrph1 {
    color:grey; background: black;
}
Class selector

It used the class attribute of an element. To select an element write a period(.) character followed by an id name.

Example: <p class="prgrph2">This is a second paragraph.</p> .prgrph2 {
    border: 1px dotted blue; margin: 10px auto;
}
Pseudo class selector

It is used to define a special state for an element. It can be use to change a style once it hover, focus or any other state.

Example: a:hover {
    color: yellow;
}

:hover is a pseudo-class selector, used to select elements when you mouse over them.

Other pseudo-class selector

:focus :nth-child(n) :disabled :not(selector) :required
Pseudo element selector

It is used to style specific parts of an element.

Example: p::first-letter {
    font-size: 3em; font-style: italic;
}

::first-letter is a pseudo-element, used to add a special style to the first letter of a text.

Other pseudo-element selector

::after ::before ::first-line ::marker ::selection
Attribute selector

Style HTML elements using specific attributes or attribute values.

  1. [attribute] selector - used to select elements with a specific attribute.a[target] {
        background-color: black;
        color: white;
    }
  2. [attribute="value"] selector - used to select elements with a specified attribute and value.div[class="main-container"] {
        border: 5px double pink;
        padding: 10px;
    }
  3. [attribute~="value"] selector - used to select elements with an attribute value containing a specified word. Attribute that contains a space-separated list of words.[class~="container"] {
        background-color: violet;
    }
  4. [attribute|="value"] selector - used to select elements with the specified attribute, whose value can be exactly the specified value, or the specified value followed by a hyphen (-).[class|="value"] {
        border-bottom: 1px solid red;
    }
  5. [attribute^="product"] selector - used to select elements with the specified attribute, whose value starts with the specified value.[class^="value"] {
        background-color: grey;
        padding: 15px;
    }
  6. [attribute$="value"] selector - used to select elements whose attribute value ends with a specified value.[class$="value"] {
        border-bottom: 1px solid red;
    }
  7. [attribute*="container"] selector - used to select elements whose attribute value contains a specified value.[class*="container"] {
        padding: 20px;
    }
